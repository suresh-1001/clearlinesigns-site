# Clear Line Signs — Website

**Live site:** https://www.clearlinesigns.com  
**Blog:** https://blog.clearlinesigns.com (separate Hugo repo)

## File Structure
```
index.html        → Homepage
services.html     → Services
portfolio.html    → Portfolio
contact.html      → Contact & quote form
thank-you.html    → Form submission confirmation
CNAME             → Custom domain (clearlinesigns.com)
assets/
  css/main.css    → Shared styles
  js/shared.js    → Shared nav + footer
  images/logo.png → Brand logo
```

## GitHub Pages Setup
Settings → Pages → Source: main branch → / (root)

## Form Backend
Uses Web3Forms (access key already configured in contact.html and index.html).
Submissions go to info@clearlinesigns.com.
